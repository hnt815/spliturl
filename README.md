## 安装

~~~shell
npm i spliturl
~~~

## 使用

~~~javascript
const url = 'https://www.google.com/home?a=b&b=c'
const result = splitUrl(url)
console.log(result)
//result[url] = https://www.google.com/home
//result[params] = {a: 'b',b: 'c'}
~~~
